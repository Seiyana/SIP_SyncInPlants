Setup Instructions

Hardware Setup

  1. Flash the ESP32 with the provided firmware
  2. Connect to "SoilMonitor_Setup" WiFi network
  3. Configure your WiFi credentials at 192.168.4.1
  4. Insert the soil moisture sensor into your plant's soil

Software Setup

  1. Deploy this web app to your hosting platform
  2. Update script.js with your Supabase credentials
  3. Calibrate the sensor in Settings
  4. Select your plant type for accurate monitoring

About

Soil Moisture Monitoring System

A complete IoT solution for monitoring soil moisture levels in real-time. This system uses an ESP32 microcontroller with a capacitive moisture sensor to collect data and send it to the cloud for analysis and visualization.

System Components

  • ESP32 Microcontroller - Reads sensor data and transmits to cloud
  • Capacitive Soil Moisture Sensor - Measures soil moisture levels
  • Supabase Backend - Real-time database and API
  • Web Dashboard - Live monitoring and historical analysis

Features

  • Real-time moisture monitoring with live updates
  • Historical data visualization with interactive charts
  • Plant-specific monitoring with custom thresholds
  • Sensor calibration for accurate readings
  • Smart notifications when plants need attention
  • Responsive design for mobile and desktop

How It Works

  1. ESP32 reads moisture sensor every 3 seconds
  2. Data is sent to Supabase via REST API
  3. Supabase broadcasts updates to connected clients
  4. Web dashboard receives and visualizes data in real-time
  5. System checks thresholds and sends notifications
