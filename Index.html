Setup Instructions

Step 1: Power On Your Sensor

  1. Connect the ESP32 microcontroller to a power source using a USB cable
  2. Insert the soil moisture sensor into your plant's soil
  3. Wait 10 seconds for the device to initialize

Step 2: Connect to WiFi

  1. On your phone or computer, look for a WiFi network named "SyncInPlants_Setup"
  2. Connect to the network
  3. Enter your WiFi credentials and save
  4. Wait 30 seconds for the device to connect

Step 3: Add the Website

  1. Visit SyncInPlants in your web browser
  2. Click "Select Plant" to choose or add your plant
  3. Calibrate your sensor in the Settings
  4. Your sensor data should now appear on the dashboard in real-time

Troubleshooting

  • If sensor not connecting: Restart the ESP32 and try the WiFi setup again
  • If no data appears: Check that notifications are enabled in Settings
  • For calibration issues: Run the calibration process again from Settings

About

Soil Moisture Monitoring System

A complete IoT solution for monitoring soil moisture levels in real-time. This system uses an ESP32 microcontroller with a capacitive moisture sensor to collect data and send it to the cloud for real-time visualization.

System Components

  • ESP32 Microcontroller - Reads sensor data and transmits to cloud
  • Capacitive Soil Moisture Sensor - Measures soil moisture levels
  • Supabase Backend - Real-time database and API
  • Web Dashboard - Live monitoring and historical analysis

Features

  • Real-time moisture monitoring with live updates
  • Historical data visualization
  • Plant-specific monitoring with custom thresholds
  • Sensor calibration for accurate readings
  • Smart notifications when plants need attention
  • Responsive design for mobile and desktop

Technology Stack

ESP32 Arduino Supabase PostgreSQL JavaScript Chart.js HTML5 CSS3
