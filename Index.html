About

üå± Soil Moisture Monitoring System

A complete IoT solution for monitoring soil moisture levels in real-time. This system uses an ESP32 microcontroller with a capacitive moisture sensor to collect data and send it to the cloud for analysis and visualization.

üîß System Components

  • ESP32 Microcontroller - Reads sensor data and transmits to cloud
  • Capacitive Soil Moisture Sensor - Measures soil moisture levels
  • Supabase Backend - Real-time database and API
  • Web Dashboard - Live monitoring and historical analysis

üìä Features

  • Real-time moisture monitoring with live updates
  • Historical data visualization with interactive charts
  • 24-hour statistics and trend analysis
  • Easy WiFi setup through captive portal
  • Responsive design for mobile and desktop
  • Customizable refresh rates and display options

üí° How It Works

  1. ESP32 reads moisture sensor every 5 seconds
  2. Data is sent to Supabase via REST API
  3. Supabase broadcasts updates to connected clients
  4. Web dashboard receives and visualizes data in real-time

ESP32 Arduino Supabase PostgreSQL JavaScript Chart.js HTML5 CSS3
